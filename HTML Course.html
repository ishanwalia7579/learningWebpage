HTML Tags

HTML consists of a series of elements that tell the browser how to display content. Here are some common HTML tags:

  1. <h1> - Heading 1
  2. <p> - Paragraph
  3. <a> - Anchor (link)
  4. <img> - Image
  5. <ul> - Unordered List
  6. <ol> - Ordered List
  7. <form> - Form
